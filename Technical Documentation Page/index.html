HTML and Basic Structure

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

It is the building block of web development and provides the basic structure of a webpage.

<html></html>

The opening and closing HTML tags are essential for the document's structure.

Within the <html> tag, you will find other important tags such as <head> and <body>.

<head></head>

The <head> tag contains meta information like the title of the page, links to stylesheets, etc.

Below is a basic HTML structure:

<html><head><title>Website</title></head><body></body></html>
  • <html>: Defines the document type
  • <head>: Contains metadata
  • <title>: Specifies the title of the page
  • <body>: Contains the content of the page
  • <h1>: Defines the main header
Styling with CSS

CSS (Cascading Style Sheets) is used to style the appearance of HTML elements.

CSS allows you to change the color, layout, and positioning of elements.

body { background-color: #f4f4f4; }

This example changes the background color of the page.

You can also use CSS to style fonts, add margins and padding, and much more.

h1 { font-family: 'Poppins', sans-serif; font-size: 36px; }

Here, we're changing the font of the h1 header to 'Poppins'.

  • Font-family: Specifies the font of an element
  • Background-color: Sets the background color
  • Padding: Adds space inside an element
  • Margin: Adds space outside an element
  • Color: Sets the color of text
Responsive Design

Responsive web design ensures that your website looks good on all devices, such as smartphones, tablets, and desktops.

@media screen and (max-width: 600px) { body { font-size: 14px; } }

The above CSS rule adjusts the font size on small screens.

Responsive design uses media queries to adapt the layout to different screen sizes.

By using flexible grid layouts, you can ensure that your website is mobile-friendly.

  • Media queries: Apply styles based on device characteristics
  • Viewport: Defines the area of the screen used by the webpage
  • Flexible grid layouts: Makes the design more adaptive
  • Relative units: Allows better scaling of elements
  • Breakpoints: Defines screen sizes at which design changes occur
Adding Interactivity with JavaScript

JavaScript is a programming language that allows you to add interactivity to your website.

You can use JavaScript to handle events such as button clicks, form submissions, and more.

document.getElementById("myButton").onclick = function() { alert('Button clicked!'); }

The above code shows how you can attach an event listener to a button.

  • Event listeners: Trigger functions based on user interactions
  • DOM manipulation: Allows you to interact with the HTML elements dynamically
  • Functions: Encapsulate code for reusability
  • Variables: Store data and values for use in your scripts
  • Loops: Iterate over collections of items
Publishing Your Website

Once you've developed your website, the next step is to publish it so others can see it.

There are many hosting services available where you can upload your files.

ftp://yourwebsite.com

FTP (File Transfer Protocol) allows you to transfer files from your local computer to the server.

You can also use cloud hosting services like GitHub Pages or Netlify for free hosting.

  • FTP: File transfer between local and remote servers
  • Web hosting: Services to store your website files
  • Cloud hosting: Host websites without physical servers
  • GitHub Pages: A free hosting service for static websites
  • Domain name: The address of your website on the internet
